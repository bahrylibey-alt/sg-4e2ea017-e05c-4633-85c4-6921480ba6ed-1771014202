## Deploy to Vercel

### 1. Push to GitHub
```bash
git add .
git commit -m "feat: complete affiliate marketing autopilot system"
git push origin main
```

### 2. Deploy on Vercel

**Option A: Via Dashboard**
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your GitHub repository
3. Configure environment variables (see below)
4. Click "Deploy"

**Option B: Via CLI**
```bash
npm install -g vercel
vercel login
vercel --prod
```

### 3. Configure Environment Variables

Add these in Vercel Dashboard → Settings → Environment Variables:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

**Important:** The `CRON_SECRET` is automatically generated by Vercel - no manual setup needed!

## Troubleshooting

### Cron Jobs Not Running?

**Check 1: Verify vercel.json**
```json
{
  "crons": [
    {
      "path": "/api/cron/discover-products",
      "schedule": "0 2 * * *"
    }
  ]
}
```

**Check 2: Environment Variables**
- Ensure SUPABASE_SERVICE_ROLE_KEY is set
- Check Vercel Dashboard → Settings → Environment Variables

**Check 3: Function Logs**
- Go to Vercel Dashboard → Functions
- Click on cron function
- Review execution logs

### Products Not Displaying?

**Check 1: Database**
```sql
SELECT COUNT(*) FROM generated_content WHERE status = 'published';
-- Should return 12
```

**Check 2: RLS Policies**
```sql
SELECT * FROM pg_policies WHERE tablename = 'generated_content';
-- Should have 'public_read_published_content' policy
```

**Check 3: Network Errors**
- Open browser console (F12)
- Check for Supabase connection errors
- Verify NEXT_PUBLIC_SUPABASE_* env vars

### Redirect Links Breaking?

**Check 1: Slug Match**
```sql
SELECT 
  gc.title,
  SUBSTRING(gc.body FROM '/go/([^)]+)') as content_slug,
  al.slug as link_slug
FROM generated_content gc
LEFT JOIN affiliate_links al ON SUBSTRING(gc.body FROM '/go/([^)]+)') = al.slug
WHERE gc.status = 'published';
-- All rows should have matching slugs
```

**Check 2: Original URLs**
```sql
SELECT network, original_url FROM affiliate_links WHERE original_url NOT LIKE 'https://%';
-- Should return 0 rows
```
